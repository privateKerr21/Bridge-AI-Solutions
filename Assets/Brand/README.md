# Bridge AI Solutions - Brand Assets

This folder contains brand-related assets and the landing page for Bridge AI Solutions.

## Landing Page

The main landing page files are located in **this folder**:
- `index.html` - Main landing page HTML
- `styles.css` - Landing page stylesheet

### Landing Page Features
- **Splash Screen**: Animated logo intro on page load
- **Scroll Animations**: Fade-in/out effects as you scroll
- **Brutalist Design**: Consistent box-style design with thick borders and offset shadows
- **Responsive**: Mobile-friendly layout
- **Color Scheme**: Black, white, and sage green accent
