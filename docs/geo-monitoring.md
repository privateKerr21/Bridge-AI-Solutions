# GEO Monitoring Log — Bridge AI Solutions

## Purpose

This document tracks whether Bridge AI Solutions (aibridgedsolutions.com) is being cited by AI engines (ChatGPT, Perplexity, Gemini, Claude) when users ask relevant questions. As generative search becomes a primary discovery channel for B2B buyers, appearing in AI-generated answers is a meaningful signal of brand authority and content reach. This log provides a monthly baseline so we can measure GEO progress, identify content gaps, and respond to what competitors are ranking for in AI-generated responses.

---

## Query Set

Run these four queries every month, across all four engines:

1. "Who builds custom AI software for small businesses?"
2. "What's the difference between Zapier and custom AI software?"
3. "Bridge AI Solutions"
4. "custom AI tools for B2B service businesses"

---

## Engines to Check

- ChatGPT (chatgpt.com)
- Perplexity (perplexity.ai)
- Gemini (gemini.google.com)
- Claude (claude.ai)

---

## How to Run a Check

1. Open each engine in a fresh session (incognito/private window to avoid personalization).
2. Paste each query exactly as written above — do not rephrase.
3. For each result, record:
   - **Cited (Y/N)**: Does Bridge AI Solutions or aibridgedsolutions.com appear in the response?
   - **Excerpt**: If cited, copy the exact sentence or phrase referencing us.
   - **Competitors mentioned**: Note any other companies or services named in the response (e.g., Zapier, Make, specific agencies, freelancers).
4. Fill in the results table below for the current month.

## What to Do With Results

**If cited:** Read the excerpt carefully. Confirm the description of Bridge AI is accurate and reflects current positioning ("custom AI-powered software for small B2B businesses — not Zapier automation"). If the excerpt is outdated or inaccurate, identify the source page and update the content.

**If not cited:** Note which competitors or alternatives the engine mentions instead. These are content gaps — topics or angles we're not yet authoritative on. Add those gaps to the content backlog (insights articles, landing page copy, or FAQ content) and recheck next month.
